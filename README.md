This repo contains the code for "Feature-based Low-Rank Compression of Large Language Models via Bayesian Optimization"

## Quick start
### Installation
```
pip install -r requirement.txt
```

### Low-rank compression
```
bash run.sh
```

### Post-training
```
bash run_posttrain.sh
```
